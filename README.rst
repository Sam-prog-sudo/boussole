========
Boussole
========

Aims to reproduce the useful Compass commandline tool behavior for 'build' and 'watch' actions.

This is under construction, not even an Alpha for now.

Requires
********

* `Watchdog`_ == 0.8.3;

Install
*******

First install the package ::

    pip install boussole
